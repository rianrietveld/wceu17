Demo aria-label

Without aria-label:
edit 

    
        <a href="your-url">
        edit
        </a>

With aria-label:
edit 


  <a href="your-url"
    aria-label="edit: my awesome post">
        edit
  </a>

But beware: aria-label overrides link text:
delete 

    
    <a href="your-url"
    aria-label="edit: my awesome post">
        delete
    </a>

Or don't use ARIA
edit: my awesome post 

    
    <a href="your-url">
         edit<span class="screen-reader-text">: my awesome post</span>
    </a>
