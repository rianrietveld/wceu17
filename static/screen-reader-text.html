Demo screen-reader-text class

Empty link:

    
        <a href="https://twitter.com/WCEurope" class="fa fa-twitter"></a>

Link text hidden with the .screen-reader-text class:
WCEU on Twitter 

    
        <a href="https://twitter.com/WCEurope" class="fa fa-twitter">
            <span class="screen-reader-text">WCEU on Twitter</span>
        </a>

Hide form label from sight: 

    
        <label for="s" class="screen-reader-text">Search</label>
        <input id="s" type="text">

        <input type="submit" value="Search">

Add an extra heading for semantics, adding context or convenience:

House pets

  • Cats
  • Dogs
  • Rabbits
    
        <h2 class="screen-reader-text">House pets</h2>
        <ul>
            <li>Cats</li>
            <li>Dogs</li>
            <li>Rabbits</li>
        </ul>

CSS screen-reader-text

        
            .screen-reader-text {
                border: 0;
                clip: rect( 1px, 1px, 1px, 1px );
                clip-path: inset( 50% );
                height: 1px;
                margin: -1px;
                overflow: hidden;
                padding: 0;
                position: absolute;
                width: 1px;
                word-wrap: normal;
            }

            .screen-reader-text:focus {
                clip: auto;
                clip-path: none;
                display: block;
                left: 0.5em;
                top: 0.5em;
                width: auto;
                z-index: 100000; /* Above WP toolbar. */
            }
